# Drupal-coder — бекенд контактной формы

Серверный рендер сайта Drupal-coder на учебном PHP-фреймворке (без сторонних
фреймворков). Контактная форма (Имя, Email, Телефон, Компания, Сообщение)
работает через единую точку входа как веб-сервис: POST создаёт запись и
возвращает логин/пароль/адрес профиля, PUT по динамическому адресу редактирует
запись после Basic-авторизации. JSON и XML поддерживаются. Без JavaScript форма
отправляется обычным POST.

## Установка

1. Создать таблицу в БД:

       mysql -u USER -p DBNAME < schema.sql

2. Скопировать конфиг и вписать пароль БД:

       cp settings.sample.php settings.php

   В settings.php при необходимости задать basedir, например
   '/WebProjectBackend/backend', если приложение лежит в подпапке.

## Маршруты

- `GET  ?q=`               — главная страница с формой
- `POST ?q=application`    — создать запись (форма, JSON или XML)
- `GET  ?q=application/{id}` — профиль/редактирование (Basic auth)
- `PUT  ?q=application/{id}` — сохранить изменения (Basic auth)
